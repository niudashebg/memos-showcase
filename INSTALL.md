# Memos Showcase - 快速安装指南

## 5 分钟快速开始

### 步骤 1：打包插件（1 分钟）

在命令行中执行：

```bash
cd wordpress-plugin
zip -r ../memos-showcase.zip .
```

或在文件管理器中：
1. 选中 `wordpress-plugin` 文件夹内的所有文件
2. 右键 → 压缩为 ZIP
3. 命名为 `memos-showcase.zip`

---

### 步骤 2：上传插件（1 分钟）

#### 方法 A：WordPress 后台上传

1. 登录 WordPress 后台
2. 进入 **插件 → 安装插件**
3. 点击 **上传插件** 按钮
4. 选择 `memos-showcase.zip`
5. 点击 **立即安装**
6. 安装完成后点击 **启用插件**

#### 方法 B：FTP 上传

1. 使用 FTP 客户端连接服务器
2. 导航到 `wp-content/plugins/`
3. 上传 `memos-showcase` 文件夹
4. 在 WordPress 后台启用插件

---

### 步骤 3：配置服务器（1 分钟）

1. 在 WordPress 后台左侧菜单找到 **Memos**
2. 点击进入设置页面
3. 在 **服务器设置** 标签页：
   - 填写 **Memos 服务器地址**
   - 例如：`http://192.168.1.100:5230`
4. 点击 **保存设置**

---

### 步骤 4：自定义样式（1 分钟）

1. 切换到 **样式设置** 标签页
2. 设置你喜欢的颜色：
   - 主色调：`#667eea`（紫色）
   - 辅助色：`#764ba2`（深紫）
3. 选择显示样式：**卡片式**
4. 选择字体：**系统默认**
5. 点击 **保存设置**

---

### 步骤 5：使用短代码（1 分钟）

#### 在文章中

1. 编辑文章
2. 添加"短代码"区块
3. 输入：
   ```
   [memos_showcase]
   ```
4. 发布文章

#### 在页面中

1. 创建新页面
2. 添加短代码：
   ```
   [memos_showcase limit="5" style="card"]
   ```
3. 发布页面

---

## ✅ 验证安装

### 检查插件是否启用

1. 进入 **插件** 页面
2. 确认 **Memos Showcase** 显示为"已启用"

### 检查设置页面

1. 点击左侧菜单 **Memos**
2. 应该能看到设置页面和 6 个标签页

### 测试短代码

1. 创建测试页面
2. 添加短代码 `[memos_showcase]`
3. 查看页面，应该能看到备忘录列表

## ❓ 常见问题

### 问题：找不到 Memos 菜单

**解决：**
1. 确认插件已启用
2. 清除浏览器缓存
3. 刷新页面

### 问题：设置页面空白

**解决：**
1. 检查 PHP 版本（需要 7.4+）
2. 检查 WordPress 版本（需要 5.0+）
3. 查看错误日志

### 问题：短代码不生效

**解决：**
1. 确认短代码拼写正确：`[memos_showcase]`
2. 确认在文本模式下添加（不是可视化模式）
3. 检查是否添加了引号（不需要引号）
