# Memos WordPress 集成 - 5 分钟快速开始

## 目标

在 5 分钟内，让你的 WordPress 博客显示 Memos 公开备忘录。

---

## 前置要求

- [ ] WordPress 网站
- [ ] Memos 服务已部署
- [ ] 有公开的备忘录（Visibility = PUBLIC）

---

## 快速步骤

### 步骤 1：上传文件（2 分钟）

#### 方法 A：FTP 上传

1. 连接 FTP
2. 上传 `wordpress-plugin` 文件夹到：
   ```
   /wp-content/plugins/memos-widget/
   ```

#### 方法 B：文件管理器

1. 登录主机控制面板
2. 进入文件管理器
3. 导航到 `wp-content/plugins/`
4. 上传 `memos-widget` 文件夹

---

### 步骤 2：激活插件（1 分钟）

1. 登录 WordPress 后台
2. 进入 **插件**
3. 找到 **Memos Widget**
4. 点击 **启用**

---

### 步骤 3：配置（1 分钟）

1. 进入 **设置 → Memos Widget**
2. 填写：
   ```
   Memos 服务器地址：http://你的NAS_IP:5230
   ```
3. 点击 **保存设置**

---

### 步骤 4：使用（1 分钟）

#### 在文章中使用

编辑文章，添加短代码：

```
[memos]
```

#### 在页面中使用

创建新页面，添加短代码：

```
[memos limit="10" style="card"]
```

---

### 步骤 5：完成！

访问文章/页面，查看效果。

编辑主题 CSS：

```css
.memo-item {
    border-left: 4px solid #0070a8;
    background: #f9f9f9;
}
```

### 修改颜色

```css

    background: #e3f2fd;
    color: #1976d2;
}
```

## 常见问题快速解决

### 问题：显示空白

**解决：**
1. 检查服务器地址是否正确
2. 确认有 PUBLIC 备忘录
3. 查看浏览器控制台错误

### 问题：样式混乱

**解决：**
1. 清除浏览器缓存
2. 检查主题 CSS 冲突

### 问题：数据不更新

**解决：**
```
设置 → Memos Widget → 减少缓存时间
```
